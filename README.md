# ✨ Sobre o Projeto

A **SprintMax** foi criada para simular uma loja online moderna voltada ao universo esportivo, trazendo uma interface elegante, intuitiva e totalmente responsiva.

O projeto possui foco em:
- experiência visual premium;
- organização profissional;
- responsividade;
- UI moderna;
- componentes reutilizáveis;
- layout limpo e elegante.
